# 操作方式和规则

## 经典操作模式
经典操作模式继承自Winmine，是兼容性最好、最权威的操作模式。简单地说有三点：左键在弹起时触发，右键在按下时触发，双击在弹起时触发。具体实现如下图：

```automaton
stateDiagram-v2
    默认状态 --> 左键按下: 左键按下
    默认状态 --> 触发标雷: 右键按下

    左键按下 --> 触发左键: 左键弹起
    触发左键 --> 默认状态

    左键按下 --> 双键按下: 右键按下

    触发标雷 --> 默认状态: 右键弹起
    触发标雷 --> 双键按下: 左键按下

    双键按下 --> 触发双键(右键): 左键弹起
    双键按下 --> 触发双键(左键): 右键弹起

    触发双键(右键) --> 默认状态: 右键弹起
    触发双键(左键) --> 默认状态: 左键弹起

    触发双键(右键) --> 双键按下: 左键按下
    触发双键(左键) --> 双键按下: 右键按下
```

### 1.5-click
1.5-click是利用右键和双键触发时机的区别实现的快速操作方法。标雷和双击常常是连在一起操作的，其中标雷需要一次点击，双击需要一次点击。如果两次点击都是一次按下和一次弹起，一共需要四次按下/弹起。在经典操作模式中，可以先按下右键标雷，然后不松右键，继续按下左键，然后两键松开完成双击，就可以通过三次按下/弹起完成标雷和双击。鼠标有硬件限制，按下和弹起都要消耗时间，所以三次按下/弹起相当于用1.5次点击时间完成了两次点击，因而得名。

实际操作中这个收益主要取决于右键按下到双键松开的时间差，这个时间差内，鼠标需要完成从标雷位置到双击位置的移动，因此不同的操作水准在1.5-click的用时上会有较大差异。极限情况下，右键按下后左键立刻按下，然后双键弹起，只消耗一次点击的时间。

## 其他操作模式
### 左键双击
很多新兴的扫雷软件和平台采用左键双击的操作模式。左键双击只有两个触发点：左键弹起和右键按下。和经典操作模式的区别是，左键弹起在未打开的格子上则触发左键，左键弹起在已打开的格子上则触发双键。左右键是独立的，因此操作难度更低。不同的平台对于左右键同时按下的处理不一致，因此这里不放具体的实现图示。

### 触屏操作
触屏操作相当于只有左键，因此如何实现右键是个问题。有的软件提供一个按钮进行左右键切换，有的软件用长按区分左右键。

### 滑动点击
滑动点击是最激进的一种操作模式。顾名思义，从一键按下开始，经过的格子都会触发，直到这一键弹起。因为它根本性地改变了扫雷操作的体验，很少有软件支持滑动点击。

### 辅助双击
辅助双击是由软件代替用户完成双击操作。软件会识别那些标够雷的数字，并自动触发双击甚至递归触发双击。这个功能同样改变了扫雷操作的体验。它让用户的注意力更多地放在逻辑而不是操作上，因此相比于滑动点击更受欢迎。使用递归双击的软件代表有Minesweeper Go、Chocolate Sweeper和Meowsweeper。
