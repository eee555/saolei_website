# 扫雷软件

我们目前支持的扫雷录像格式有`EVF`、`RMV`、`AVF`、`MVF`。其中`EVF`和`RMV`是通用扫雷录像格式，`AVF`是Minesweeper Arbiter专属，`MVF`是Minesweeper Clone专属。

如果你希望开源扫雷网支持更多的扫雷软件，你可以选择生成`EVF`或`RMV`，或者让ms-toollib支持你的自定义格式。

## 录像准入门槛
出于公平的原因，开源扫雷网只接收特定规则约束下的扫雷录像。

### 不支持的非传统功能
- 不支持左键双击
- 不支持递归双击
- 录像必须有完整的鼠标轨迹，不支持触屏录像

### 支持的非传统功能
- 将鼠标锁定在边框内。这个功能最初由Minesweeper Arbiter支持，会大大提高初级成绩。现在元扫雷也支持该功能。虽然这个功能破坏了公平性，但是实际上却提高了初级的竞技性和趣味性。另一方面，用某些其他软件（例如[Magpie](https://github.com/Blinue/Magpie)）本身就可以实现将鼠标锁定在任何矩形区域的功能，所以你也可以在其他软件中将鼠标锁定在边框内。

## 常见问题

### 为什么不支持触屏录像？
我们知道扫雷的触屏玩家数量庞大，但是触屏录像对于识别作弊者产生了很大障碍。触屏录像无法包含手的移动情况，只能看到光标来回跳动，和机器人操作难以区分。鼠标轨迹是识别操作是否是真人的重要依据。

### 为什么不支持左键双击？
历史上，扫雷的双击一直都是左右键同时点击，左键双击是为了兼容触屏扫雷发明的功能。在鼠标上使用左键双击，相比于传统的左右键同时点击，操作难度更低，容错更高，因此在同一个排行榜不公平。
