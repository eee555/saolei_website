# 金羊杯

金羊杯是由[郭锦洋](https://openms.top/#/player/48)举办的，历史悠久的赛事。比赛于每年的4月和10月举办，为期6天。

## 计分规则

金羊杯会统计比赛期间上传的录像。

成绩取以下录像：

- 初级：成绩最好的 20 局，要求 `3BV >= 10`
- 中级：成绩最好的 12 局，要求 `3BV >= 30`
- 高级：成绩最好的 5 局，要求 `3BV >= 100`

如果局数不够，会用默认成绩补足：

| 级别 | 补足成绩 |
| --- | --- |
| 初级 | 10 秒 |
| 中级 | 60 秒 |
| 高级 | 240 秒 |

最终排名以这些录像的总成绩为基础。成绩越小，排名越靠前。

## 接受的软件

### 元扫雷

直接用元扫雷的“比赛标识”功能即可参赛。

### Minesweeper Arbiter

比赛进行中时，需要在比赛详情页注册一个 Arbiter 参赛标识。

参赛标识必须以比赛页面给出的比赛标识结尾。例如页面给出的标识是 `G12345`，则 Arbiter 标识需要包含这个结尾。

注册成功后，使用该 Arbiter 标识上传的录像会被识别为比赛录像。
