# 比赛功能

比赛功能用于在限定时间内收集参赛录像，并在比赛结束后展示成绩、排名和录像包。目前网站主要支持[金羊杯](./gsc.md)比赛。如果你有心举办比赛，可以联系开发者添加新的比赛模式。

## 进入比赛页面

从主站顶栏进入 **比赛** 页面。比赛首页会显示比赛列表，包括：

- 比赛状态
- 比赛名称
- 主办方
- 开始时间
- 结束时间

点击比赛列表中的任意一行，可以打开该比赛详情页。

## 比赛状态

比赛可能处于以下状态：

| 状态 | 含义 |
| --- | --- |
| 审核中 | 比赛还没有确认完整赛程，暂时不能正常参赛。 |
| 即将开始 | 比赛已经确认赛程，但尚未到开始时间。 |
| 进行中 | 比赛正在进行，可以按页面提示上传参赛录像。 |
| 结算中 | 比赛已结束，系统正在统计成绩。 |
| 已结束 | 成绩已经结算，可以查看最终排名。 |
| 已取消 | 比赛不再进行。 |

## 如何参赛

比赛详情页会在比赛即将开始或进行中时显示参赛说明。不同比赛模式、不同软件的参赛方式可能不同。金羊杯的参赛方式见[金羊杯](./gsc.md#如何参赛)。

## 查看即时成绩

比赛进行中时，登录用户可以在比赛详情页查看自己的即时成绩。根据比赛模式不同，页面可能展示：

- 概览
- 录像列表
- Bv 表
- 管理功能

即时成绩可以手动刷新。由于录像处理和统计需要时间，刚上传的录像不一定立刻出现在比赛页面。

## 查看最终结果

比赛结束并完成结算后，详情页会显示最终排名。

在排名表中点击某位选手，可以查看该选手的比赛录像和成绩概览。

## 下载录像

比赛页面提供录像下载功能：

- 下载全部录像：在最终结果页中下载整场比赛录像包。
- 下载个人录像包：在选手详情的管理页中下载该选手的比赛录像。

这些下载接口有限流限制，页面会提示对应的频率限制。

## 比赛录像与普通录像

被识别为正在进行的比赛录像后，这些录像会暂时从普通录像队列、个人纪录和部分排行统计中排除。

比赛结束后，系统会把不再属于进行中比赛的录像恢复为普通录像，并刷新相关队列和纪录。

这意味着比赛期间上传的录像不一定立刻影响普通个人纪录；比赛结算后，相关记录会重新纳入正常统计。
